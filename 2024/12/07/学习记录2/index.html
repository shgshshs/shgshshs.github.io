str_replace()

它的作用是替换字符串中的某些内容，具体来说，它可以将目标字符串中的所有匹配项替换为指定的替换内容。

函数原型：

参数说明：

  1. **$search**：需要被替换的字符串（可以是单个字符、字符串或数组）。
  2. **$replace**：用来替换 $search 的字符串（也可以是一个数组）。
  3. **$subject**：要进行替换的目标字符串（或者是字符串数组）。
  4. **$count**（可选）：传入一个变量用于接收返回的替换次数。

返回值：

返回替换后的字符串或者替换后的字符串数组。

preg_match()

用于执行正则表达式匹配的函数。它用于检查一个字符串是否匹配给定的正则表达式。如果匹配成功，它会返回 1，否则返回 0。如果发生错误，返回 FALSE。

语法：

参数：

  1. $pattern: 正则表达式的模式，通常以/开始和结束，例如：/abc/。

  2. $subject: 要进行匹配的字符串。

  3. $matches (可选): 用来保存匹配结果的数组。它会包含匹配的字符串，如果有分组的话，数组的第一个元素是完整的匹配结果，后续元素是捕获组的内容。

    输出：

  4. $flags

    (可选): 额外的匹配选项。常见的选项有：

    • PREG_OFFSET_CAPTURE: 如果设置此标志，$matches数组将包含每个匹配的偏移量（即匹配字符串的起始位置）。

  5. $offset (可选): 搜索的起始位置。默认值为 0。

返回值：

  • 如果匹配成功，返回 1。
  • 如果匹配失败，返回 0。
  • 如果出现错误，返回 FALSE。

intval（）

该函数用于将一个值转换为整数类型（int）。这个函数会返回给定值的整数表示。它可以接受字符串、浮动数值、布尔值等类型的输入，并返回相应的整数。

语法：

参数：

  • $var: 需要转换为整数的变量。它可以是字符串、数字、布尔值、数组或对象。
  • $base (可选): 数值转换的基数，默认是 10，表示十进制。如果 base 是 0，则通过检测 var 的格式来决定使用的进制。base 参数允许你指定不同的进制，如二进制（2）、八进制（8）、十六进制（16）等。

返回值：

  • 返回转换后的整数值。如果无法转换，则返回 0。
